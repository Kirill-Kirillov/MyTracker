    REST - server MyTracker создан в первую очередь для практического применения имеющихся знаний в области spring
    Данный сервер позволяет создавать проект и вести учет временных (duration) либо колличественных показателей(quantity), потраченных на проект
    1. Для работы данного сервера необходима БД PostgreSql
    1.1 Для создания таблиц, необходимых для сервера можно воспользоваться файлом tables.sql
    2. Необходимо настроить подключение в файле application.properties (1-3 строки)
